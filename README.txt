[EN] ENGLISH (UK)
--------------------------------------------------------------------------------

1. Project Overview
   - Project Name: Elysium λ Development & Research
   - Client / Organisation: Elysium Development & Research
   - Delivery Date: March 2026 (Continuous Development)
   - Site Version: 1.2.0
   - Author / Developer Studio: Daniel Morales / Elysium λ
   - Site Objective: To provide professional web development and research services 
     for SMEs in Europe and Latin America, blending software engineering with 
     applied research.

2. Brand Identity
   - Logo: The Lambda (λ) symbol, representing research, logic, and efficiency.
   - Logo Versions: 
     - Main: Blue Lambda with Platinum text.
     - Dark Background: Platinum Lambda / Blue Glow.
     - Favicon: Multi-size PNG/SVG logo.
   - Minimum Size: 16px (favicon), 24px (UI icon).
   - Safety Space: 25% of symbol width.
   - Incorrect Uses: Color modification outside the palette, stretching, 
     removing the symbol from "ELYSIUM" in primary branding.
   - Brand Tone: Formal, modern, minimalist, technical, and reliable.
   - Visual Values: Precision, transparency, and innovation.

3. Color System
   - Primary Colors: #020410 (Deepest Midnight Blue), #E1E1E5 (Platinum White).
   - Secondary Colors: #051025 (Rich Navy), #A0A5B5 (Muted Blue-Grey).
   - Neutral Colors: #6B7280 (Dark Slate), #1a1a1a (Background variations).
   - Accent Colors: #2997ff (Electric Blue), #D4AF37 (Subtle Gold).
   - Background Colors: Variable midnight gradients (Liquid Alloy system).
   - Text Colors: #E1E1E5 (Primary), #A0A5B5 (Secondary).

4. Typography System
   - Font Families:
     - Primary (Headings): -apple-system, BlinkMacSystemFont, "SF Pro Display", "Segoe UI", Roboto...
     - Secondary (Body): -apple-system, BlinkMacSystemFont, "SF Pro Text"...
     - Fallback: Arial, sans-serif.
   - Font Weights:
     - Light (200), Regular (400), Medium (500), Bold (600/700).
   - Font Scale:
     - H1: 4rem (Hero 5rem) / Line-height: 1.1 / Letter-spacing: -0.03em.
     - Body: 1.05rem / Line-height: 1.6 / Letter-spacing: 0.

5. Layout System
   - Grid System: 12-column flex/grid system.
   - Container Width: 1200px.
   - Breakpoints:
     - Mobile: < 480px.
     - Tablet: < 768px.
     - Laptop: < 1100px.
     - Desktop: > 1200px.

6. Spacing System
   - xs: 0.5rem (8px)
   - sm: 1rem (16px)
   - md: 2rem (32px)
   - lg: 4rem (64px)
   - xl: 8rem (128px)

7. UI Components
   - Buttons:
     - Primary: Metallic gradient (#e1e1e5 to #c0c0c5), dark text, white hover glow.
     - Secondary/Ghost: Transparent with 1px glass border, silver glow on hover.
   - Forms:
     - Input fields: 16px radius, dark glass bg, electric blue focus border.
     - Error messages: Subtle red indicators with high contrast.
   - Cards:
     - Layout: Padding 3rem, radius 32px, glassmorphism blur (20px).
     - Shadow: Subtle depth with metallic edge reflections.
   - Navigation:
     - Navbar: Fixed glass pill-shape, language switcher, mobile toggle (☰).

8. Responsive Design
   - Desktop/Laptop: Full 12-column reveal, sticky headers.
   - Tablet: Multi-column grids (2-column cards), simplified navigation.
   - Mobile: Single-column stacking, hidden side navigation (hamburger menu).
   - Adaptive Images: object-fit: cover, lazy loading.

9. Media Guidelines
   - Format: WebP (primary), PNG (transparency), SVG (icons).
   - Sizes: Hero images (1920px+), Web previews (800px).
   - Compression: Heavy lossless compression for sub-second loads.
   - Ratio: 16:9 for banners, 1:1 for symbols/logos.

10. Technical Stack
    - Frontend: HTML5, Vanilla CSS3 (Custom Variables), Vanilla JavaScript (ES6+).
    - Hosting: Cloudflare Workers / Pages.
    - Domain: IONOS (elysiumdr.eu).
    - Version Control: GitHub.

11. Folder Structure
    - /es, /pt: Language-specific directories.
    - /CSS: Design system and page styles.
    - /JS: Core interactivity and scroll logic.
    - /Images: Branded assets and project previews.
    - /CV: Professional profile documentation.

12. Deployment Information
    - Hosting: Cloudflare Pages for edge-level performance.
    - CDN: Global Cloudflare Edge network.
    - Build Process: Direct GitHub deployment via Cloudflare CI/CD.

13. Performance Guidelines
    - Minimal external libraries.
    - Native CSS animations over JS.
    - Background "Liquid Alloy" uses GPU-accelerated gradients.

14. Accessibility
    - Semantic HTML5 structure.
    - ARIA labels for language switchers and toggles.
    - Focus-visible states for keyboard navigation.

15. Maintenance Instructions
    - Update content in language-specific HTML files.
    - Manage styles in base.css for global changes.
    - Use Cloudflare Wrangler for local testing.

16. License / Usage
    - Rights: All rights reserved to Elysium λ Development & Research.
    - Modification: Authorised personnel only (Daniel Morales).

17. Support
    - Technical Contact: Daniel Morales
    - Email: info@elysiumdr.eu

18. Motion Guidelines
    - Animations: Apple-style reveals on scroll, Liquid Alloy background flow.
    - Duration: 0.2s (fast), 0.4s (medium), 12s/24s (bg loop).
    - Easing: cubic-bezier(0.16, 1, 0.3, 1) or cubic-bezier(0.25, 0.1, 0.25, 1).

19. Iconography
    - Style: Minimalist, linear, high-contrast.
    - Blue/Gold gradients for primary icons.

20. Design Principles
    - Minimalism: Focus on negative space and core content.
    - Clarity: High readability and clear navigation paths.
    - Hierarchy: Strong typographical contrast to guide the eye.
