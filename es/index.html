Metodología científica

01

Preguntar

¿Cuáles son las fortalezas, las oportunidades, las debilidades y las amenazas de su empresa para identificar el potencial de esta?

02

Preparar

Se recopilarán los datos necesarios para poder crear una arquitectura del proyecto.

03

Analizar

Con todos los datos recopilados se procede a crear un diseño inicial de la aplicación web.

04

Procesar

Con el esquema, los datos y la visión consolidada, desarrollamos el esqueleto y la estructura técnica de la aplicación web.

05

Desarrollar

Durante esta parte el código se desarrolla en profundidad, con todas las extensiones y buenas prácticas de Clean Code.

06

Publicar

Una vez la página haya superado todas las pruebas de usabilidad y de experiencia de usuario, se lanzará la primera versión oficial al público.
